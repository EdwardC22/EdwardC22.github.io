# EdwardC22.github.io

个人网站项目，使用 React + TypeScript + Vite + Tailwind CSS 构建。

## 📦 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建项目
npm run build

# 预览构建结果
npm run serve
```

## ⚙️ 技术栈

- **框架**: React 18
- **语言**: TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **国际化**: react-i18next
- **图标**: FontAwesome, Heroicons
